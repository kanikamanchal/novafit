# 💪 NovaFit - Gym Website

## 📌 Description
NovaFit is a responsive gym website built using HTML, CSS, and JavaScript. It provides a modern and user-friendly interface where users can explore gym services, workout sections, and fitness-related content.

---

## ✨ Features
- Responsive design (mobile, tablet, desktop)
- Modern and attractive UI
- Gym services and workout sections
- Smooth navigation
- Interactive elements using JavaScript

---

## 🧰 Technologies Used
- HTML5  
- CSS3  
- JavaScript
